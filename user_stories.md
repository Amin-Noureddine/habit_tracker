# Habit Tracker App - User Stories

## 1. Login / Registration

---

## User Story: Account Registration

**Title:** Account Registration

As a user, I want to register with my name, username, age, and country so that I can create an account and access the habit tracking features.

### Acceptance Criteria:
1. User can enter name, username, age, and country.
2. System validates required fields.
3. User account is created successfully.

**Priority:** High

**Story Points:** 5

**Labels:**
- enhancement
- priority: high

---

## User Story: Account Login

**Title:** Account Login

As a user, I want to log in using my username and password so that I can access my account and track my habits.

### Acceptance Criteria:
